===========
Create Group
===========

Create a Crafter Studio group.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/group/create.json``                      |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Admin                                                            |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| groupname    || String     || |checkmark|  || Groupname to use                                |
+---------------+-------------+---------------+--------------------------------------------------+
|| description  || String     || |checkmark|  || Group description                               |
+---------------+-------------+---------------+--------------------------------------------------+
|| site         || String     || |checkmark|  || Parent site                                     |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

.. code-block:: json

	POST .../api/1/services/api/1/group/create.json

.. code-block:: json

	{
		"groupname" : "contributors",
		"description" : "Content Contributors",
		"site" : "mysite"
	}

--------
Response
--------

+---------+---------------------------------------------+---------------------------------------------------+
|| Status || Location                                   || Response Body                                    |
+=========+=============================================+===================================================+
|| 201    || ``.../group/get.json?username=:groupname`` || ``{ "status" : "OK" }``                          |
+---------+---------------------------------------------+---------------------------------------------------+
|| 401    ||                                            || ``{ "status" : "Unauthorized" }``                |
+---------+---------------------------------------------+---------------------------------------------------+
|| 409    || ``.../group/get.json?username=:groupname`` || ``{ "status" : "Group already exists" }``        |
+---------+---------------------------------------------+---------------------------------------------------+
|| 500    ||                                            || ``{ "status" : "Internal server error" }``       |
+---------+---------------------------------------------+---------------------------------------------------+

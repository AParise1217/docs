===========
Enable User
===========

Enable a Crafter Studio user.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/user/enable-user.json``                  |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Admin                                                            |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| username     || String     || |checkmark|  || Username to use                                 |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

.. code-block:: json

	POST .../api/1/services/api/1/user/enable-user.json

.. code-block:: json

	{
		"username" : "jane.doe"
	}

--------
Response
--------

+---------+-------------------------------------------+---------------------------------------------------+
|| Status || Location                                 || Response Body                                    |
+=========+===========================================+===================================================+
|| 200    || ``.../get-user.json?username=:username`` || ``{ "status" : "OK" }``                          |
+---------+-------------------------------------------+---------------------------------------------------+
|| 401    ||                                          || ``{ "status" : "Unauthorized" }``                |
+---------+-------------------------------------------+---------------------------------------------------+
|| 500    ||                                          || ``{ "status" : "Internal server error" }``       |
+---------+-------------------------------------------+---------------------------------------------------+

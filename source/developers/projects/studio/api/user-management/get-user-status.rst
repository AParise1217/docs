===============
Get User Status
===============

Get the status of a Crafter Studio user.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/user/get-user-status.json``              |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Admin                                                            |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| username     || String     || |checkmark|  || Username to use                                 |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

.. code-block:: json

	GET .../api/1/services/api/1/user/get-user-status.json?username=jane.doe

.. code-block:: json

	{
		"username" : "jane.doe",
		"enabled" : "true"
	}

--------
Response
--------

+---------+-------------------------------------------+---------------------------------------------------+
|| Status || Location                                 || Response Body                                    |
+=========+===========================================+===================================================+
|| 200    || ``.../get-user.json?username=:username`` || ``{ "status" : "OK" }``                          |
+---------+-------------------------------------------+---------------------------------------------------+
|| 401    ||                                          || ``{ "status" : "Unauthorized" }``                |
+---------+-------------------------------------------+---------------------------------------------------+
|| 404    ||                                          || ``{ "status" : "User not found"}``               |
+---------+-------------------------------------------+---------------------------------------------------+
|| 500    ||                                          || ``{ "status" : "Internal server error" }``       |
+---------+-------------------------------------------+---------------------------------------------------+

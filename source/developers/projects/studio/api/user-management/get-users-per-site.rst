==================
Get Users per Site
==================

Get Crafter Studio users per site with an optional range for pagination.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/user/get-users-per-site.json``           |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Admin, site member                                               |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| site         || String     || |checkmark|  || Site to look under for users                    |
+---------------+-------------+---------------+--------------------------------------------------+
|| start        || Integer    ||              || Start at this count in the result set           |
+---------------+-------------+---------------+--------------------------------------------------+
|| end          || Integer    ||              || End at this count in the result set             |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

.. code-block:: json

	GET .../api/1/services/api/1/user/get-users.json?site=mySite

.. code-block:: json

  {
    "users" :
      [
        {
          "username" : "jane.doe",
          "first_name" : "Jane",
          "last_name" : "Doe",
          "email" : "jane@example.com",
        },
        {
          "username" : "joe.bloggs",
          "first_name" : "Joe",
          "last_name" : "Bloggs",
          "email" : "joe@example.com",
        }
    ]
  }

--------
Response
--------

+---------+----------------------------------------+---------------------------------------------------+
|| Status || Location                              || Response Body                                    |
+=========+========================================+===================================================+
|| 200    || ``.../get-users.json?site=mySite``    || See example above.                               |
+---------+----------------------------------------+---------------------------------------------------+
|| 401    ||                                       || ``{ "status" : "Unauthorized" }``                |
+---------+----------------------------------------+---------------------------------------------------+
|| 404    ||                                       || ``{ "status" : "Site not found"}``               |
+---------+----------------------------------------+---------------------------------------------------+
|| 500    ||                                       || ``{ "status" : "Internal server error" }``       |
+---------+----------------------------------------+---------------------------------------------------+

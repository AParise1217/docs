========
Get User
========

Get a Crafter Studio user.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/user/get-user.json/:username``           |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Admin, self, member in same project                              |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| username     || String     || |checkmark|  || Username to use                                 |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

.. code-block:: json

	GET .../api/1/services/api/1/user/get-user.json/jane.doe

--------
Response
--------

+---------+----------------------------------+---------------------------------------------------+
|| Status || Location                        || Response Body                                    |
+=========+==================================+===================================================+
|| 200    || ``.../get-user.json/:username`` || ``{ "status" : "OK" }``                          |
+---------+----------------------------------+---------------------------------------------------+
|| 401    ||                                 || ``{ "status" : "Unauthorized" }``                |
+---------+----------------------------------+---------------------------------------------------+
|| 404    ||                                 || ``{ "status" : "User not found" }``              |
+---------+----------------------------------+---------------------------------------------------+
|| 500    ||                                 || ``{ "status" : "Internal server error" }``       |
+---------+----------------------------------+---------------------------------------------------+

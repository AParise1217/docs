==========
Box Upload
==========

Upload a file to box.

.. NOTE::
  Make sure that the box profile id to be used has been configured.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/box/upload.json``                        |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || N/A                                                              |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| site         || String     || |checkmark|  || Site to use                                     |
+---------------+-------------+---------------+--------------------------------------------------+
|| profile      || String     || |checkmark|  || Profile Id to use                               |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------
^^^^^^^
Request
^^^^^^^

``POST ...api/1/services/api/1/box/upload?site=myawesomesite&profile=box_default``

.. NOTE::
    This request needs to be sent with ``Content-Type=multipart/form-data`` and the file as parameter ``file``.


^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

.. code-block:: json

  {"name":"708213662.mp4","id":"268775475280"}

---------
Responses
---------

+---------+-------------------------------------------+---------------------------------------------------+
|| Status || Location                                 || Response Body                                    |
+=========+===========================================+===================================================+
|| 200    ||                                          || See example above.                               |
+---------+-------------------------------------------+---------------------------------------------------+
|| 400    ||                                          || ``{ "message" : "Bad request" }``                |
+---------+-------------------------------------------+---------------------------------------------------+
|| 500    ||                                          || ``{ "message" : "Internal server error" }``      |
+---------+-------------------------------------------+---------------------------------------------------+

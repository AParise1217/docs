==========
Get Target
==========

Get a Crafter Deployer target.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/target/:target_id/get.json``                            |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+-------------------------+-------------+---------------+----------------------------------------+
|| Name                   || Type       || Required     || Description                           |
+=========================+=============+===============+========================================+
|| target_id              || String     || |checkmark|  || The target ID                         |
+-------------------------+-------------+---------------+----------------------------------------+

-------
Example
-------

.. code-block:: json

	POST .../api/1/target/mysite/get.json

.. code-block:: json

  {
    "target_id" : "mysite",
    "load_date" : "2017-01-26T10:00:1.234",
    "configuration_file" : "/opt/crafter/deployer/conf/mysite.yaml"
  }

--------
Response
--------

+---------+-------------------------------------+-------------------------------------------------------+
|| Status || Location                           || Response Body                                        |
+=========+=====================================+=======================================================+
|| 200    || ``.../target/:target_id/get.json`` || See example above.                                   |
+---------+-------------------------------------+-------------------------------------------------------+
|| 500    ||                                    || ``{ "message" : EXCEPTION_MESSAGE }``                |
+---------+-------------------------------------+-------------------------------------------------------+

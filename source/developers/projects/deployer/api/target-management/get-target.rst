==========
Get Target
==========

Get a Crafter Deployer target.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/target/get/:target_id``                                 |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+-------------------------+-------------+---------------+----------------------------------------+
|| Name                   || Type       || Required     || Description                           |
+=========================+=============+===============+========================================+
|| target_id              || String     || |checkmark|  || The target ID                         |
+-------------------------+-------------+---------------+----------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

``POST .../api/1/target/get/:target_id``

^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

.. code-block:: json

  {
    "target_id" : "mysite",
    "load_date" : "2017-01-26T10:00:1.234-05:00"
  }

---------
Responses
---------

+---------+-------------------------------------+-------------------------------------------------------+
|| Status || Location                           || Response Body                                        |
+=========+=====================================+=======================================================+
|| 200    || ``.../target/get/:target_id``      || See example above.                                   |
+---------+-------------------------------------+-------------------------------------------------------+
|| 404    ||                                    || ``{ "message" : "Target not found" }``               |
+---------+-------------------------------------+-------------------------------------------------------+
|| 500    ||                                    || ``{ "message" : "Internal server error" }``          |
+---------+-------------------------------------+-------------------------------------------------------+

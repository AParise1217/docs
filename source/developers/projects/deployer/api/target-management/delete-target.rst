=============
Delete Target
=============

Delete a Crafter Deployer target.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/target/:target_id/delete.json``                         |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+-------------------------+-------------+---------------+----------------------------------------+
|| Name                   || Type       || Required     || Description                           |
+=========================+=============+===============+========================================+
|| target_id              || String     || |checkmark|  || The target ID                         |
+-------------------------+-------------+---------------+----------------------------------------+

-------
Example
-------

.. code-block:: json

	POST .../api/1/target/mysite/delete.json

.. code-block:: json

  {
    "target_id" : "mysite"
  }

--------
Response
--------

+---------+----------------------------------+---------------------------------------------------+
|| Status || Location                        || Response Body                                    |
+=========+==================================+===================================================+
|| 204    ||                                 ||                                                  |
+---------+----------------------------------+---------------------------------------------------+
|| 404    ||                                 || ``{ "message" : "Target not found" }``           |
+---------+----------------------------------+---------------------------------------------------+
|| 500    ||                                 || ``{ "message" : EXCEPTION_MESSAGE }``            |
+---------+----------------------------------+---------------------------------------------------+
